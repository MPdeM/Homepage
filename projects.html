Deep Learning

  • Covid X-Ray image classification

    Deep learning techniques for image analysis and classification can be used on X-ray images to spot anomalies and aid healthcare personnel in the diagnosis. Based on publicly available data of Covid-19 positive and normals this model was able to properly predict the risk of Covid-19 on a X-ray image.

    The project incorporates:

    • data selection from Kaggle and other sources
    • data inspection - actually a really important part
    • data processing: dataset construction, preprocessing (color and shape), normalization, and buil the train and tests sets
    • data modeling: CNN using KerasTensor Flow in CoLab
    • data validation and Prediction
    • deploiment: check with real patient images

    Github
